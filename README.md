## 프로젝트 소개

**Trafi**는 사용자의 취향을 분석해 최적의 여행지를 제안하는 개인 맞춤형 여행 추천 플랫폼입니다.
단순한 위치 기반 추천을 넘어, 사용자의 라이프 스타일과 성향을 반영한 ‘축제 MBTI’ 진단을 통해 개개인에게 꼭 맞는 축제와 여행지 조합을 제안하는 것이 Trafi의 목적입니다.
